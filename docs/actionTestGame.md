# "Action Test" Sample Game #

The purpose of this sample game is to quickly be able to create most types
of action results that are currently possible in a game of 1E.

## Setup cards ##

### Hands ###

Player 1:
* Metaphasic Shields - can be downloaded by Dr. Reyga and Host Metaphasic Shielding Test
* Starfleet Type II Phaser - can be downloaded by Beverly
* Assign Mission Specialists
* Cowboy Diplomacy
* Gorath x3 - one to report for free with Legitimate Leader; two to use as cost for Get It Done

Player 2:
* Tomalak - a [Cmd] icon personnel for the dilemma Shall We Begin Again?
* Mot the Barber because why not

### Draw Decks ###

P1 only:
* Metaphasic Shields x2
* Starfleet Type II Phaser x2
* Mission specialists: Mot the Barber, Simon Tarses, Palteth, Dr. Farek x2, Kromm

### Discard ###

P1 only:
* Togaran - to be placed on deck by Diplomatic Contact
* Ambassador Spock - to be placed in point area by Scientific Diplomacy
* Taitt x2 - filler

### Core ###

P1 only:
* Continuing Mission - to get [TNG] icon
* Legitimate Leader of the Empire
* Get It Done

### At Locations ###

Mission 1 - Visit Tranquil Colony (shared)
* Away Team on planet: Sarek, Jean-Luc Picard, Giusti, Riva, Darian Wallace
* Federation Outpost w/ Beverly aboard
* Stolen Ship
* Seeded dilemmas in order of encounter:
  * Tense Negotiations
  * Punishment Box

Mission 2 - Control Plague
* Away Team on planet: Gowron, Korris, Narik x2
* Klingon Outpost
* Yridian Shuttle w/ Devinoni Ral + Koral aboard
* Diplomatic Contact is on Devinoni Ral
* Seeded dilemmas in order of encounter:
  * Armus: Skin of Evil
  * Shall We Begin Again?
  * New Essentialists
  * Flim-Flam Artist

Mission 3 - Host Metaphasic Shielding Test
* Scientific Diplomacy is on the mission
* I.K.S. K'Vort w/ Dr. Reyga, Klag, and K'mpec aboard
* no dilemmas

## Recommended actions ##

### Card play phase ###

* Play Assign Mission Specialists. Download any mission specialists you want to outpost of choice.
* Play Cowboy Diplomacy for free to score points.
* Play Gorath for free using Legitimate Leader of the Empire. Report him wherever.
* Use special downloads if you want to. (Not using them will be a good test of how invasive the "Do you want to perform this action?" prompts are.)
Total points: 5

### Execute orders phase ###

* Change affiliations back and forth for Koral and Stolen Ship
* Move around with Yridian Shuttle: dock, undock, beam/walk cards to and from outposts
* Use Diplomatic Contact to retrieve Togarath from discard; Diplomatic Contact is discarded
* Use Get It Done to boost a card's attributes (don't use it to retrieve discard cards if you want to test retrieving Ambassador Spock later)
Total points: 5

### Mission 1: Host Metaphasic Shielding Test ###

Attempt the mission. If you are using the crew that was setup on the I.K.S. K'Vort, you will
immediately solve it. This will trigger:
* Scoring 40 points from the mission
* Scoring 5 points with Klag + AMS
* Option to score points from a SCIENCE in discard using Scientific Diplomacy; use Ambassador Spock for 10 points
* Option to download Metaphasic Shields to the K'Vort using the mission's gametext
Total points: 60

### Mission 2: Control Plague ###

Attempt the mission. This will trigger
a number of random selection dilemmas. Volunteer for any with Korris to test the volunteer action.
* Armus: Skin of Evil - kill a personnel
* Shall We Begin Again? - reveal opponent's hand; stops a personnel
* New Essentialists - places a personnel in draw deck
* Flim-Flam Artist - option for opponent to draw a card; should stop entire Away Team and fail the mission
Total points: 60

### Mission 3: Visit Tranquil Colony ###

If you attempt the mission using the 5-person Away Team from setup:
* Tense Negotiations - nullified by Sarek
* Punishment Box - select Giusti to be stopped so Picard can help solve the mission
The Away Team can solve the mission. Either Darian or Riva will score 5 points from AMS.
Total points: 100

The game should end at this point since victory conditions have been met by player 1.
